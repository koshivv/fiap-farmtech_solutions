# 🌱 FarmTech Solutions

FarmTech Solutions é um projeto acadêmico focado no desenvolvimento de soluções simples de **agricultura digital utilizando programação e análise de dados**.

O projeto integra **Python** e **R** para simular um pequeno sistema de gerenciamento agrícola aliado a análises estatísticas e dados climáticos.

---

# 📌 Objetivo do Projeto

O objetivo deste projeto é demonstrar a integração entre:

* **Desenvolvimento em Python**
* **Manipulação e análise de dados**
* **Estatística aplicada com R**
* **Consumo de APIs externas**

Tudo isso aplicado ao contexto de **tecnologia no agronegócio**.

---

# ⚙️ Tecnologias Utilizadas

* **Python** – desenvolvimento do sistema CRUD via terminal
* **R** – análise estatística e consumo de API meteorológica
* **CSV** – armazenamento de dados de teste
* **API Meteorológica** – obtenção de dados climáticos

---

# 🧩 Funcionalidades

## 🖥️ Sistema em Python

O sistema principal é um **CRUD com interação via terminal**, permitindo:

* Criar registros
* Visualizar dados
* Atualizar informações
* Remover registros

Esse sistema simula o gerenciamento de informações utilizadas no planejamento agrícola.

---

## 📊 Scripts em R

O projeto também inclui scripts desenvolvidos em **R**, responsáveis por:

### Análise Estatística

Realização de cálculos estatísticos a partir dos dados fornecidos pelo sistema.

### Integração com API Meteorológica

Realiza requisições a uma API externa para coletar **dados climáticos**, que podem ser utilizados para análises relacionadas à produção agrícola.

# 🚀 Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/seu-usuario/farmtech_solutions.git
```

### 2️⃣ Acessar a pasta do projeto

```bash
cd farmtech_solutions
```

### 3️⃣ Executar o sistema em Python

```bash
python python/crud_farmtech.py
```

### 4️⃣ Executar os scripts em R

Abra os arquivos `.R` em um ambiente como **RStudio** ou execute via terminal:

```bash
Rscript r/analise_estatistica.R
```
